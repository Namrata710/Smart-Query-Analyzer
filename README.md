# Smart-Query-Analyzer
# Performance Analysis of Database Query Execution Using JDBC with and without Indexing

This project analyzes database query execution performance using JDBC before and after applying indexing.

## Tech Stack
- Java
- JDBC
- MySQL

## How to Run
1. Create database and table using SQL script.
2. Compile Java files.
3. Run QueryPerformance.java
